# Pokédex - React Native

This project is a mobile Pokédex application developed with **React Native** and **TypeScript**.  
It is an educational project based on a tutorial from the YouTube channel: Grafikart.  
It uses the public [PokeAPI](https://pokeapi.co/) to display, filter, and navigate a Pokémon database in a smooth and interactive way.

---

## Objective of the Project

The primary purpose of this project is educational, aiming to improve mobile development skills, including:

- Practicing mobile development with **React Native**
- Deepening the use of **TypeScript** in a real-world project
- Mastering **data fetching** with **TanStack Query**
- Using **React Reanimated** for performant animations
- Implementing a **modular architecture** based on reusable components

---

## Main Features

- **Search** Pokémon by **name** or **number**
- **Sort** Pokémon by **alphabetical order** or by **ID number**  
  ![image](https://github.com/user-attachments/assets/cbcb0d9c-075c-456c-b635-a65621183a09)
- **Detailed display** of a Pokémon on a dedicated screen, with styles that adapt based on the Pokémon type
- Navigation between Pokémon using **chevrons**
- Smooth interface with **React Reanimated animations**
 ![image](https://github.com/user-attachments/assets/aa1dc234-af1b-40a3-a2e7-946a7c728066)

- Centralized management of **colors**, **data fetching**, and **UI components**

---

## Tech Stack

- React Native (Expo)
- TypeScript
- React Reanimated
- TanStack Query (formerly React Query)
- React Navigation
- PokeAPI

---

## Getting Started

> Requires a working React Native environment (via Expo or native CLI)
